Anleitung

Sie brauchen ein schwarz-weisses Bild im pbm Format. Z.B. kann ImageMagick soziemlich jedes Format nach pbm konvertieren:

convert MeinBild MeinBild.pbm

Leider gibt es verschiedene pbm Formate, so dass mit

pbmb_to_pbma

konvertiert werden kann. Der Quellcode ist hier kopiert von hier. Wenn die nötigen Comiler da sind, so sollte

make pbmb_to_pbma

alles Notwendige leisten. Nach dem Konvertieren muss dan leider noch die Kommentarzeile entfernt werden. Dann kann mit

python visuelle_krypto.py

von hier die Magie erstellt werden.
